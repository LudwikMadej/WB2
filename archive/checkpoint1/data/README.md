# Data Directory

This directory stores all intermediate and final data artifacts for the Checkpoint 1 experiment.

## Subdirectories

- **images/**: Local storage for resized (224x224) CelebA images in JPG format.
- **activations/**: High-dimensional activation vectors extracted from the CLIP model.

## Files

- **metadata.csv**: A central registry mapping image filenames to labels ("bald") and dataset splits ("train" for CAV training, "test" for recovery analysis).
